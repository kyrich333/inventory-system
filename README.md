# Inventory Management System (PERN Stack)

A scalable inventory management system designed to manage multiple item types, brands, and locations.

## Features

- Multi-location inventory
- Multi-brand item management
- Stock movement tracking
- Filtering and search
- Scalable architecture

## Tech Stack

- PostgreSQL
- Express.js
- React
- Node.js

## Architecture

Client → API → Database
React → Express → PostgreSQL
